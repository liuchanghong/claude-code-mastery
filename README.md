>
> 不是官方文档的翻译。不是入门教程的堆砌。

---

## 这本书是给谁的

- 你会写代码，但不一定是 AI 专家
- 你听说过 Claude Code，但还没发挥出它的真正威力
- 你讨厌那种"AI 很厉害，以下是 10 个技巧"的废话文章
- 你希望读完一本书就能真正改变工作方式

**你的背景不重要。**
iOS 开发者、后端工程师、全栈、脚本仔——只要你写代码，这本书就适合你。

---

## 如何阅读这本书

**不用从头读到尾。** 这本书按你的需求设计：

| 你的情况 | 从这里开始 |
|---------|-----------|
| 刚安装 Claude Code | 第 1 章 → 第 2 章 → 第 3 章 |
| 用了一段时间但感觉没发挥威力 | 第 4 章 → 第 7 章 → 第 10 章 |
| 想在 iOS/Swift 项目里用 | 第 9 章（可单独阅读）|
| 想彻底改造工作流 | 第 11 章 → 第 13 章 |
| 遇到具体问题 | 直接查附录 |

每章开头都有 **TL;DR**，5 秒钟知道这章值不值得你读。
